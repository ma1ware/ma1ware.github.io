Jade模板引擎

Feb 7, 2017

jade是一个由javascript实现的高性能的模板引擎，支持Node、php、scala、ruby、python、java等。

安装jade
npm install jade

新建一个test.jade文件

然后到终端执行jade -P test.jade后就可以得到格式化后的test.html

这样可以大大加快我们的开发速度
