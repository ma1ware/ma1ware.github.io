Feb 15, 2017

环境搭建

Flask是python开发的一套轻量级并且非常灵活的web框架，首先搭建开发环境

安装python，在三个平台下的安装可以登录Python官网选择对应的版本
可以安装venv来创建虚拟环境，以免造成本机python包混乱，安装可以通过pip pip install virtualenv
venv\Scripts\activate 激活环境后安装flask，(venv) $ pip install flask
